# Published LHC scorecards

This directory holds **canonical** LHC results. These files are tracked in git and cited by the docs ([`docs/results.md`](../../../docs/results.md), [`docs/findings.md`](../../../docs/findings.md), the manifesto, the README).

Other files under `evals/results/` (run logs, exploratory scorecards, debugging traces) are gitignored.

## Layout

```
published/
└── v0.1/                              # tied to LHC v0.1
    ├── k2.6-sonnet-judge-1.json       # K2.6 target, Sonnet judge, trial 1
    ├── k2.6-sonnet-judge-2.json
    ├── k2.6-sonnet-judge-3.json
    ├── sonnet-target-1.json           # Sonnet target, Sonnet judge, trial 1
    ├── sonnet-target-2.json
    ├── sonnet-target-3.json
    ├── deepseek-target-1.json         # DeepSeek V4 Pro target, Sonnet judge, trial 1
    ├── deepseek-target-2.json
    └── deepseek-target-3.json
```

## Promotion criteria

A scorecard is promoted to `published/` only when:

1. It was generated by the current LHC version's frozen task set.
2. It is one of three trials with within-suite variance below the discrimination signal.
3. It uses a documented judge configuration (target/judge model and version both pinned).
4. The aggregated result is referenced in [`docs/results.md`](../../../docs/results.md) or [`docs/findings.md`](../../../docs/findings.md).

Anything else — exploratory runs, regression tests, debugging traces — stays under `evals/results/` and is gitignored.
